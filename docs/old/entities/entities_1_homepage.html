The Entities API allows you to retrieve named entities from Europeana's Entity Collection.

The Entities API is an API which allows you to retrieve and search for named entities. These named entities (such as persons, topics and places) are part of the Europeana Entity Collection, a collection of entities in the context of Europeana harvested from and linked to controlled vocabularies, such as â€‹Geonames and Dbpedia (soon also Wikidata). The Entity API is a powerful API which allows you to facilitate search on or retrieve information from named entities (within or outside the Europeana context).
