Reusability parameter

The possible values of the reusability parameters.

Value Description
open The records are freely reusable. The licenses in this category are:
  • PD
  • NOC
  • CC0
  • CC BY
  • CC BY-SA
restricted The records are reusable, but with restrictions. The licenses in this category are:
  • CC BY-NC
  • CC BY-NC-SA
  • CC BY-NC-ND
  • CC BY-ND
  • CC OOC-NC
  • InC-EDU
  • NoC-NC
  • NoC-OKLR
permission You can reuse the records only with explicit permission. The licenses in this category are:
  • RR-F
  • RR-P
  • RR-R
  • Unknown
  • RS InC
  • RS InC-OW-EU
  • RS CNE

Search only for freely reusable records:

&query=Paris&reusability=open
