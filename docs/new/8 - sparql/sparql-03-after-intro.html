Update

The SPARQL API is powered by a separate database (Virtuoso) which reflects the current state of the Europeana datasets as in the main Europeana APIs and is regularly updated.

Credits

The initial pilots were set up by Ontotext under the framework of the Europeana Creative project and using the Ontotext GraphDB semantic repository (which has been replaced by Virtuoso).
